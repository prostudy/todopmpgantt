============================================================
  EJEMPLO: RED DE ACTIVIDADES CON 2 RECURSOS
  Ruta Critica, Holguras y Costeo de Proyecto
============================================================

ESCENARIO:
----------
Un proyecto tiene 5 actividades con las siguientes relaciones
de precedencia y duraciones (en semanas):

  Actividad | Precedencia | Duracion (semanas)
  ----------|-------------|-------------------
  Inicio    | --          | 0
  1         | Inicio      | 3
  2         | 1           | 3
  3         | 1           | 6
  4         | 2           | 8
  5         | 3, 4        | 4
  Fin       | 5           | 0

Se cuenta con 2 recursos:
  - Recurso 1: cuesta $10 por dia ($50/semana)
  - Recurso 2: cuesta $20 por dia ($100/semana)

NOTA: En la herramienta, 1 semana = 5 dias laborales.
      Las duraciones se convierten asi: semanas x 5 = dias.

COMO IMPORTAR:
--------------
1. Abrir index.html en el navegador
2. Clic en "Importar CSV/JSON"
3. Seleccionar: red_actividades_2recursos.json
4. Clic en "Recalcular"

ASIGNACION DE RECURSOS:
------------------------
  Act 1: Recurso 1 al 100% → 15 dias x $10 = $150
  Act 2: Recurso 2 al 100% → 15 dias x $20 = $300
  Act 3: Recurso 1 al 100% → 30 dias x $10 = $300
  Act 4: Recurso 2 al 100% → 40 dias x $20 = $800
  Act 5: Recurso 1 al 50% + Recurso 2 al 50%
         → Rec1: 20 dias x $10 x 0.5 = $100
         → Rec2: 20 dias x $20 x 0.5 = $200
         → Subtotal = $300

RED DEL PROYECTO (diagrama de nodos):
--------------------------------------

  [Inicio] → [Act 1 (3sem)] → [Act 2 (3sem)] → [Act 4 (8sem)] ↘
                             ↘                                    [Act 5 (4sem)] → [Fin]
                              → [Act 3 (6sem)] ──────────────── ↗

CAMINOS POSIBLES:
-----------------
  Camino A: Inicio → 1 → 2 → 4 → 5 → Fin
            0 + 3 + 3 + 8 + 4 = 18 semanas  ← RUTA CRITICA

  Camino B: Inicio → 1 → 3 → 5 → Fin
            0 + 3 + 6 + 4 = 13 semanas

CALCULO CPM (Critical Path Method):
-------------------------------------

  Forward Pass (ES/EF en semanas):
  ┌──────────┬────┬────┐
  │ Actividad│ ES │ EF │
  ├──────────┼────┼────┤
  │ Inicio   │  0 │  0 │
  │ Act 1    │  0 │  3 │
  │ Act 2    │  3 │  6 │
  │ Act 3    │  3 │  9 │
  │ Act 4    │  6 │ 14 │
  │ Act 5    │ 14 │ 18 │  (max(EF Act3=9, EF Act4=14) = 14)
  │ Fin      │ 18 │ 18 │
  └──────────┴────┴────┘

  Backward Pass (LS/LF en semanas):
  ┌──────────┬────┬────┐
  │ Actividad│ LS │ LF │
  ├──────────┼────┼────┤
  │ Fin      │ 18 │ 18 │
  │ Act 5    │ 14 │ 18 │
  │ Act 4    │  6 │ 14 │
  │ Act 3    │  8 │ 14 │  (LF = LS de Act5 = 14)
  │ Act 2    │  3 │  6 │
  │ Act 1    │  0 │  3 │
  │ Inicio   │  0 │  0 │
  └──────────┴────┴────┘

  Holguras:
  ┌──────────┬────────────────┬────────────────┬────────┐
  │ Actividad│ Holgura Total  │ Holgura Libre  │Critica?│
  │          │ (LS - ES)      │                │        │
  ├──────────┼────────────────┼────────────────┼────────┤
  │ Inicio   │ 0              │ 0              │   SI   │
  │ Act 1    │ 0              │ 0              │   SI   │
  │ Act 2    │ 0              │ 0              │   SI   │
  │ Act 3    │ 5              │ 5              │   NO   │
  │ Act 4    │ 0              │ 0              │   SI   │
  │ Act 5    │ 0              │ 0              │   SI   │
  │ Fin      │ 0              │ 0              │   SI   │
  └──────────┴────────────────┴────────────────┘────────┘

  Ruta Critica: Inicio → 1 → 2 → 4 → 5 → Fin (18 semanas)

  Act 3 tiene holgura de 5 semanas:
  puede retrasarse hasta 5 semanas sin afectar la fecha de fin.

COSTO TOTAL DEL PROYECTO:
--------------------------
  Act 1: Recurso 1 × 15 dias × $10/dia    = $  150
  Act 2: Recurso 2 × 15 dias × $20/dia    = $  300
  Act 3: Recurso 1 × 30 dias × $10/dia    = $  300
  Act 4: Recurso 2 × 40 dias × $20/dia    = $  800
  Act 5: Rec1 (50%) 20d × $5 + Rec2 (50%) 20d × $10 = $  300
  ─────────────────────────────────────────────────────
  TOTAL (BAC)                              = $1,850

COSTO POR RECURSO:
-------------------
  Recurso 1: Act1($150) + Act3($300) + Act5($100) = $  550
  Recurso 2: Act2($300) + Act4($800) + Act5($200) = $1,300
  ──────────────────────────────────────────────────────
  TOTAL                                            = $1,850

DURACION TOTAL: 18 semanas (90 dias laborales)

PREGUNTAS DE REFLEXION:
------------------------
1. Si Act 3 se retrasa 3 semanas, cambia la ruta critica?
   (No, porque tiene 5 semanas de holgura)

2. Si Act 3 se retrasa 6 semanas, que pasa?
   (Se convierte en critica, el proyecto se retrasa 1 semana)

3. Que pasaria si el Recurso 1 solo puede trabajar al 50%?
   Act 1 duraria 6 semanas, Act 3 duraria 12 semanas.
   Nueva ruta critica? Nuevos costos?

4. Si necesitamos acortar el proyecto 2 semanas,
   que actividad conviene comprimir (crashing)?
   Considere el costo por semana de cada recurso.
